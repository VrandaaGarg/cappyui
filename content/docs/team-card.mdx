## Customization

### Team Members

Update the `TEAM_MEMBERS` array to customize the team members:

```tsx
const TEAM_MEMBERS: TeamMember[] = [
  {
    name: "Your Name",
    role: "Your Role",
    image: "https://your-image-url.com/image.jpg",
  },
  // Add 3 more members for a 2x2 grid
];
```

### Animation Timing

Adjust the scanning duration by modifying the `SCAN_DURATION` constant:

```tsx
const SCAN_DURATION = 1.6; // Duration in seconds for each member
```

You can also fine-tune the timing synchronization between the scanning border and card effects:

```tsx
// Border movement timing
const FRAME_KEYFRAME_TIMES = [0, 0.07, 0.25, 0.32, 0.5, 0.57, 0.7, 0.8, 1];
// Card effects timing (slightly delayed for better synchronization)
const CARD_KEYFRAME_TIMES = [0, 0.07, 0.25, 0.32, 0.5, 0.57, 0.85, 0.9, 1];
```
