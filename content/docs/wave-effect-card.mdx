## Customization

### Custom Card Labels

Provide your own array of card labels:

```tsx
<WaveEffectCard
  items={[
    "Button 1",
    "Action",
    "Settings",
    "Profile",
    "Dashboard",
    "Analytics",
    "Reports",
    "Users",
  ]}
/>
```

### Wave Animation Properties

Customize the wave animation behavior:

```tsx
<WaveEffectCard
  waveSpeed={200} // Pixels per second (default: 150)
  waveThickness={120} // Ring thickness in pixels (default: 100)
  fadeWidth={60} // Fade effect width on ring edges (default: 50)
/>
```

### Opacity Settings

Adjust the base and active opacity levels:

```tsx
<WaveEffectCard
  baseOpacity={0.3} // Opacity when not in wave (default: 0.4)
  activeOpacity={1} // Opacity when in wave (default: 1)
/>
```

### Center Content

Customize the center overlay with title, subtitle, description, and icon:

```tsx
<WaveEffectCard
  title="We're serious about privacy."
  subtitle="Your data is built with privacy at its core."
  description="We never train on your data."
  icon={<YourCustomIcon />}
/>
```

## Props

| Prop            | Type             | Default                                      | Description                                         |
| --------------- | ---------------- | -------------------------------------------- | --------------------------------------------------- |
| `items`         | `string[]`       | 80 "Ripple" labels                           | Array of card labels                                |
| `waveSpeed`     | `number`         | `120`                                        | Wave animation speed in pixels per second           |
| `waveThickness` | `number`         | `60`                                         | Thickness of the wave ring in pixels                |
| `fadeWidth`     | `number`         | `30`                                         | Width of the fade effect on ring edges              |
| `baseOpacity`   | `number`         | `0.4`                                        | Card opacity when outside the wave                  |
| `activeOpacity` | `number`         | `1`                                          | Card opacity when inside the wave ring              |
| `title`         | `string`         | `"We're serious about privacy."`             | Main heading text in center overlay                 |
| `subtitle`      | `string`         | `"Your data is built with privacy..."`       | Subtitle text in center overlay                     |
| `description`   | `string`         | `"We never train on your data."`             | Description text in center overlay                  |
| `icon`          | `React.ReactNode`| Privacy lock icon                            | Custom icon for center overlay                      |
| `className`     | `string`         | `undefined`                                  | Additional CSS classes to apply to the container    |
