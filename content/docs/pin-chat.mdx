## Props

This component is designed as a self-contained demo and doesn't accept props. To customize it for your use case:

- Replace `SAMPLE_CHATS` with your own chat data source
- Modify `DEFAULT_PINNED_CHATS` to set initial pinned items
- Change `STORAGE_KEY` to use a unique localStorage key
- Customize the styling using Tailwind classes

Each chat includes a title, last message preview, and timestamp for realistic demonstration.
