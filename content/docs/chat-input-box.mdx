## Features

### Multi-Provider AI Model Selection

Choose from 17+ AI models across 5 major providers with searchable dropdown:

**Claude (Anthropic)**

- Claude Sonnet 4
- Claude Sonnet 4.5
- Claude Haiku 3.5

**Google**

- Gemini 2.5 Pro
- Gemini 2.5 Flash

**OpenAI**

- OpenAI 5
- OpenAI 5 Nano
- OpenAI 5 Mini
- OpenAI o4 Mini

**Qwen**

- Qwen3 Max
- Qwen3 235B A22B
- Qwen3 30B A3B Thinking 2507

**DeepSeek**

- DeepSeek R1 Fast
- DeepSeek V3.1

### Style Options

Choose from 5 conversation styles with descriptions:

- **Technical**: Detailed technical explanations
- **Study**: Educational and learning focused
- **Classic**: Traditional and formal tone
- **Professional**: Business and work appropriate
- **Formal**: Structured and official

### AI Tools

Toggle various AI capabilities:

- **Web Search**: Enable real-time web search
- **Reddit Search**: Search Reddit for community insights
- **Create Image**: Generate images from text descriptions

### File Upload Support

Comprehensive file handling with multiple input methods:

- **Click to Upload**: Paperclip button for traditional file selection
- **Drag & Drop**: Visual feedback when dragging files over input area
- **Clipboard Paste**: Paste images directly from clipboard (Ctrl/Cmd + V)
- **Multiple Files**: Upload multiple files simultaneously
- **Supported Types**: Images (PNG, JPG, GIF, WebP), Documents (PDF, TXT, DOC, DOCX)
- **Visual Previews**: Image thumbnails and file type icons
- **Upload Progress**: Animated progress indicators
- **Easy Removal**: Hover to remove individual files

## Keyboard Shortcuts

- **Enter**: Send message
- **Shift + Enter**: New line in message
- **Ctrl/Cmd + V**: Paste content (including images)

## Component Behavior

### Message Handling

When a message is sent (Enter key or Send button):

- Message must not be empty (whitespace trimmed)
- Console logs the message with all settings:
  - Message content
  - Selected AI model
  - Selected style
  - Tool states (web search, Reddit search, create image)
- Message input is cleared
- In production, this is where you'd make your API call

### File Upload Flow

1. User uploads file(s) via click, drag-and-drop, or paste
2. Files are processed and added to upload queue
3. Upload progress is simulated with smooth animation
4. Image files generate preview thumbnails using object URLs
5. Document files display appropriate type icons
6. Hover over files to see full filename and remove button
7. Object URLs are properly cleaned up on removal or unmount

### Dropdown Behavior

- Click outside any dropdown to close it
- Model dropdown search clears when closing or selecting
- Style dropdown is nested within Tools dropdown
- Opening one major dropdown (Model/Tools) doesn't affect the other
- Smooth animations for all dropdown interactions
