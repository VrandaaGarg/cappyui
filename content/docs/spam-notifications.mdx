### Custom Messages

You can customize the notification messages by modifying the `ALL_NOTIFICATION_MESSAGES` array:

```tsx
const ALL_NOTIFICATION_MESSAGES = [
  {
    id: "1",
    icon: Gift, // Any lucide-react icon
    title: "Custom Title",
    time: "just now",
    subtitle: "Your custom notification message here",
    color: "bg-blue-500/90", // Tailwind color with opacity
  },
  // Add more messages...
];
```
